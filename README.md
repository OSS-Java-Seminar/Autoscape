# Autoscape

<h1>Description</h1>

Autoscape is web application which enables ordinary people to rent their private cars to the users of the Autoscape application.

<h1>Technologies</h1>

<ul>
  <li>Java</li>
  <li>Springboot</li>
  <li>Boostrap</li>
  <li>Thymeleaf</li>
</ul>

<h1>Features</h1>

<ul>
  <li>Registration</li>
  <li>Login</li>
  <li>Reviews</li>
  <li>Image upload</li>
  <li>Rent a car</li>
  <li>Put a car up for rent</li>
</ul>
